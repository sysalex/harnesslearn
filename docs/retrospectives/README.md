# 阶段回顾说明

本目录用于存放每个阶段完成后的回顾文档，是 Harness Engineering L4 阶段级反馈的一部分。

## 何时创建

- 完成一个阶段后
- 重大功能完成后
- 交接前需要总结经验时

## 命名规则

推荐文件名：

```md
YYYY-MM-DD-phase-<n>-retrospective.md
```

示例：

```md
2026-04-22-phase-1-retrospective.md
```
